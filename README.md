### Cài Đặt Nhanh

```bash

# Cài đặt dependencies
npm install

npm run dev
```

## 🤝 Hướng Dẫn Đóng Góp

1. Fork dự án
2. Tạo branch tính năng (`git checkout -b feature/amazing-feature`)
3. Commit thay đổi (`git commit -m 'Add amazing feature'`)
4. Push lên branch (`git push origin feature/amazing-feature`)
5. Mở Pull Request
